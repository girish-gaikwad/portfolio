# Portfolio

A modern developer portfolio built with Next.js, featuring interactive UI, project showcases, work experience, achievements, and contact workflows.

## Tech Stack

- Framework: Next.js 13, React 18, TypeScript
- Styling: Tailwind CSS, PostCSS, Framer Motion
- 3D and Visual Effects: Three.js, React Three Fiber, Drei, React Particles
- Content: MDX, Gray Matter, Remark, Shiki
- State and Theming: Redux Toolkit, React Redux, next-themes
- Email and Integrations: Postmark, EmailJS, Vercel Analytics

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.

## Build

```bash
npm run build
npm run start
```
